## Categories

There are 3 main categories of block:

- [Application](/app/docs/blocks/application) - blocks that are used to build
  web and mobile apps
- [Marketing](/app/docs/blocks/marketing) - blocks that are used to build
  landing pages and marketing sites
- [Elements](/app/docs/blocks/elements) - smaller blocks like "Button" and
  "Image" that can be used to build other, bigger blocks
